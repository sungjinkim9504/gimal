## 가산점 항목 상세

### AI/LLM 기능 직접 구현
- **내용**: Google Gemini 1.5 Flash API를 직접 호출하여 일기 내용 기반 감정 분석 및 조언 생성
- **구현 위치**: `lib/features/ai/services/gemini_service.dart`
- **핵심 기술**:
  - Gemini REST API 직접 호출 (dio 사용)
  - system instruction을 통한 프롬프트 엔지니어링
  - JSON 응답 파싱 (감정 이모지 + 코멘트 추출)
- **증빙**: 소스코드 및 실행 화면 스크린샷
