## AI Agent 협업 방법

### 효과적인 지시 방법

| 상황 | 지시 예시 |
|------|-----------|
| 기능 추가 | "일기에 사진 첨부 기능을 추가해줘" |
| 버그 수정 | "상세 화면에서 뒤로가기 누르면 앱이 종료되는 문제 수정해줘" |
| 문서 업데이트 | "오늘 결정한 사항으로 회의록 업데이트해줘" |
| 코드 설명 | "DiaryNotifier의 updateEntry가 어떻게 동작하는지 설명해줘" |

### 매 주차 AI에게 시킬 메타 작업

1. **회의록 자동 정리** — 세션 종료 전 "오늘 결정한 것 회의록으로 정리해줘"
2. **진행률 보고서 업데이트** — "weekly-report.md 현재 상태로 업데이트해줘"
3. **질문 목록 추출** — "이번 구현에서 내가 모를 수 있는 개념 questions.md에 추가해줘"
4. **일정 체크** — "schedule.md에서 완료된 항목 체크해줘"
5. **git 커밋/푸시** — "오늘 변경사항 커밋하고 push해줘"

## git 커밋 컨벤션

```
feat:     새로운 기능 추가
fix:      버그 수정
docs:     문서 수정
refactor: 코드 리팩터링
chore:    설정, 패키지 등 기타 변경
```

**예시:**
```bash
git commit -m "feat: 일기 검색 기능 추가"
git commit -m "fix: AI 코멘트 로딩 중 화면 깜빡임 수정"
git commit -m "docs: README 및 setup.md 작성"
```
